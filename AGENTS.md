# 프로젝트 개요
이 프로젝트는 학원 스케줄 관리 웹앱를 만드는 프로젝트입니다.
예) 월-토 어떤 학원에 가는지, 유치원 픽업시간, 집 drop off시간
유치원 자녀 하원/학원 스케줄 관리용으로 밝고 따뜻한 느낌의 칸반보드 (파스텔톤 기반) 


# 만들고 싶은 기능
너무 복잡하지 않게, 한눈에 “학원명”과 “하원시간”이 가장 잘 보이게
1. 전체 화면
   배경은 연한 파스텔톤 또는 부드러운 그라데이션으로 적용
   카드와 컬럼은 둥근 모서리와 은은한 그림자 적용
   여백을 넉넉하게 주되, 모바일에서는 스크롤이 편하게 구성
2. 요일 칸반 컬럼
   월~금 컬럼은 통일된 밝은 톤
   토요일 컬럼은 평일과 다른 색상 계열로 구분
   오늘 요일 컬럼은 색상을 확실히 다르게 표시
   오늘 요일 컬럼에는 은은한 깜빡임 효과를 추가
   스케줄이 없는 요일에는 🚶 도보하원을 표시
3. 스케줄 카드

카드는 3행 구조로 정리해줘.

1행
가장 크게 표시
학원종류 아이콘 + 학원명 표시
수정/삭제 버튼은 오른쪽에 작은 아이콘으로 표시
전체를 정확히 2개 영역으로 나누고, 세로 가운데 정렬
2행
등원시간 + 하원시간 표시
하원시간을 더 강조
“등원”, “하원” 글씨 높이와 정렬을 맞춰줘
3행
하원담당 전화, 학원 전화, 홈페이지 링크를 한 줄에 표시
값이 있는 항목만 표시
한 줄을 넘기지 않게 아이콘 중심으로 간결하게 표시
하원담당은 하원, 학원전화는 학원, URL은 홈페이지로 표시
전화 아이콘 클릭 시 바로 전화 연결되게 처리
홈페이지 아이콘 클릭 시 새 창으로 열리게 처리
4. 상단 헤더
   현재 날짜와 현재 시간을 보기 좋게 표시
   스케줄 등록 버튼은 상단에 1개만 유지
   하원까지 남은 시간 표시 영역을 눈에 띄게 배치
   하원 1시간 전, 30분 전, 10분 전에는 강조 스타일을 더 크게 적용
   페이지 공유 버튼도 상단에 추가
5. 반응형
   PC에서는 요일 컬럼이 가로로 보기 좋게 배치
   모바일에서는 요일별 컬럼이 세로로 자연스럽게 쌓이도록 구성
   글자가 너무 작아지지 않게 유지
   주의사항
   기능을 새로 크게 갈아엎지 말고 기존 코드 위주로 디자인만 개선해줘
   LocalStorage 저장 구조는 가능하면 유지해줘
   초보자도 이해할 수 있도록 한국어 주석을 유지해줘
   한 번에 너무 큰 구조 변경은 하지 말고, 필요한 CSS/마크업 중심으로 단계적으로 수정해줘

# 기술/조건 (모르면 비워둬도 됨)
- HTML, CSS, JavaScript만 사용
- 별도 설치 없이 브라우저에서 바로 실행되게

# 규칙
- 코드는 초보자도 읽기 쉽게, 주석을 한국어로 달아줘
- 한 번에 너무 많이 바꾸지 말고 단계별로 진행해줘

# DB연결 관련 
현재 LocalStorage 기반 학원 스케줄 웹앱에 Firebase Firestore를 연동해줘.

가족 구성원이 동일한 데이터를 공유할 수 있도록 하고,
GitHub Pages에서 바로 동작해야 한다.

기존 UI는 유지하고,
등록/수정/삭제 기능을 Firestore 기반으로 변경해줘.

Firebase SDK는 CDN 방식으로 불러와줘.
firebaseConfig 값은 내가 나중에 붙여넣을 수 있게 빈 객체/주석으로 남겨줘.
Firestore 컬렉션명은 schedules로 사용해줘.

Firestore 연결 실패 시 LocalStorage를 fallback으로 사용해줘.
한국어 주석을 추가해줘.
